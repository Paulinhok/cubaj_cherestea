Aplicația este destinată pentru calcularea volumului de cherestea, calculând instantaneu cubajul de cherestea, în funcție de lungime. latime si grosimea lemnului. Aplicația permite crearea unei liste, asemănătoare avizului în care se pot adăuga toate măsuratorile pentru fiecare lemn cubat, lista cuprinzând grosimea, latimea, lungimea și cubajul fiecarui lemn măsurat, cubajul tuturor elementelor și pretul total calculat în functie de prețul pe metru cub.
